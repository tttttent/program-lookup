# Program Requirements Lookup

Simple static web app to browse the university program requirements CSV.

Usage
- Open `index.html` from the `program-lookup` folder in a browser.
- Or run a local HTTP server (recommended) from the folder and visit `http://localhost:8000`.

Example (Python 3):
```
cd program-lookup
python -m http.server 8000
```

Files
- `index.html` - UI
- `app.js` - CSV parsing and UI logic
- `styles.css` - styling
- `data/checklist.csv` - the dataset (from your attachment)
